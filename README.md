# mocha-vite-puppeteer

Run your Mocha tests with Vite bundler and Puppeteer.

## Install

```bash
npm install mocha-vite-puppeteer
```

## Run

First create a test.html next to your index.html.

```bash
mocha-vite-puppeteer
```
